# Projeto Global Solution 2025 - WebDev

## Descricao do projeto

O FutureSkills é uma programa web desenvolvido como parte da Global Solution 2025, com o objetivo de facilitar a inscrição de usuários em programas de capacitação profissional ligados às novas demandas do mercado de trabalho.

A aplicação disponibiliza um formulário completo onde o candidato pode preencher seus dados pessoais, selecionar sua área de interesse e registrar suas principais habilidades. O sistema realiza validações automáticas de todos os campos, garantindo que as informações inseridas estejam corretas e completas antes da confirmação da inscrição.

O projeto foi desenvolvido utilizando HTML e JavaScript puro, priorizando acessibilidade, práticas modernas de desenvolvimento e uma interface simples, funcional e intuitiva.

## Instruções de Uso

**Como testar a aplicação:**

1. Preencha o campo **Nome Completo**, **CPF** e **E-mail**.
2. Selecione um **Tipo de Interesse** (um dos rádios).
3. No campo de adicionar habilidade, digite uma habilidade e clique em **Adicionar**. Repita até ter pelo menos 3 habilidades.
4. Clique em **Finalizar Inscrição**.
5. A área de feedback (`#feedback`) exibirá erros (lista) ou a mensagem de sucesso com um resumo.

**Estrutura de arquivos importante:**

- `index.html` — marcação do formulário e elementos principais.
- `main.js` — lógica de validação, adição de habilidades e exibição de feedback.
